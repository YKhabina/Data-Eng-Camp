## 👩‍💻 О себе

Привет! Меня зовут **Юлия Хабина**, я системный аналитик с фокусом на инженерные подходы к работе с данными. 

Этот репозиторий создан для хранения учебных и рабочих материалов в рамках **Data Engineering Camp**. Здесь собираются и структурируются скрипты, заметки и лучшие практики по анализу данных, ETL-процессам и автоматизации.

**Ключевые компетенции:**
- 📊 **Системный анализ:** Выявление бизнес-требований, описание методов API, документация, UML / BPMN.
- 🐍 **Python:** Обработка данных (Pandas).
- 🗄️ **SQL:** Написание запросов
