# Проект:  Расчет метрик на базе ClickHouse для обработки событий пользователей

## Техническое задание (ТЗ)

### Цель
Реализовать расчет метрик (уникальные пользователи, сумма баллов, количество действий) 

### Функциональные требования

1. **Решение включает**  
- хранение сырых логов с TTL 30 дней;
- автоматическое агрегирование данных для трендового анализа (уникальные пользователи, сумма баллов, количество действий) с TTL 180 дней;
- расчёт удержания пользователей (retention) — доля вернувшихся в течение 7 дней после первого визита;
- запрос по дням, пользователям, сумме баллов и числу событий.
